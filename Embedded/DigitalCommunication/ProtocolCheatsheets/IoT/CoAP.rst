=====
CoAP
=====

.. note::
   This is a cheatsheet for CoAP protocol.

**CoAP**
- Constrained Application Protocol
- Lightweight, RESTful protocol for IoT
- UDP-based, low-power devices

**Key Features**
- Request/response, similar to HTTP
- Asynchronous, multicast support
- DTLS security

**Mnemonic**: "Constrained, Optimized Application Protocol"

**Common Use Cases**
- Smart home, sensors, M2M, IoT

**Pros**
- Simple, efficient, low overhead
- Designed for constrained devices

**Cons**
- Less mature than HTTP
- UDP reliability issues

**Icon**: 🌱
