=====
AMQP
=====

.. note::
   This is a cheatsheet for AMQP protocol.

**AMQP**
- Advanced Message Queuing Protocol
- Open, application layer protocol for message-oriented middleware
- Used in enterprise messaging, IoT

**Key Features**
- Reliable, secure, transactional
- Queues, topics, publish/subscribe
- Interoperable, vendor-neutral

**Mnemonic**: "Advanced Messaging, Queues, Publish!"

**Common Use Cases**
- Enterprise integration, IoT, cloud messaging

**Pros**
- Reliable delivery, flexible routing
- Open standard

**Cons**
- More complex than MQTT
- Higher resource usage

**Icon**: 📦

**Reference**: https://www.amqp.org/resources/specifications
