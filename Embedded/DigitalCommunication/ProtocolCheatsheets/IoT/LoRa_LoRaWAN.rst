============
LoRa / LoRaWAN
============

.. note::
   This is a cheatsheet for LoRa and LoRaWAN protocols.

**LoRa**
- Long Range, low-power wireless
- Chirp spread spectrum modulation
- Used for IoT, sensors, remote monitoring

**LoRaWAN**
- MAC layer protocol for LoRa
- Star-of-stars topology, gateways
- Secure, bidirectional

**Key Features**
- Long range (km), low data rate
- Low power, battery-friendly
- Adaptive data rate

**Mnemonic**: "Long Range, WAN for IoT"

**Common Use Cases**
- Smart cities, agriculture, asset tracking

**Pros**
- Wide coverage, low cost
- Open standard (LoRaWAN)

**Cons**
- Low throughput
- Duty cycle limits
