============
EtherNet/IP
============

.. note::
   This is a cheatsheet for EtherNet/IP protocol.

**EtherNet/IP**
- Industrial Ethernet protocol (CIP over Ethernet)
- Used for factory automation, process control
- Object-oriented, real-time

**Key Features**
- Uses standard Ethernet (IEEE 802.3)
- Explicit (messaging) and implicit (I/O) communication
- Scalable, integrates with IT networks

**Mnemonic**: "Ethernet for Industrial Protocols"

**Common Use Cases**
- PLCs, sensors, actuators, SCADA

**Pros**
- Open standard, wide adoption
- Leverages existing Ethernet infrastructure

**Cons**
- Not as deterministic as some fieldbuses
- Requires network segmentation for real-time
