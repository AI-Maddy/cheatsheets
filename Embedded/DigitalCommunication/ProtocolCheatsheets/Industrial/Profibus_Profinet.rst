====================
Profibus / Profinet
====================

.. note::
   This is a cheatsheet for Profibus and Profinet protocols.

**Profibus**
- Fieldbus protocol for automation (factory/process)
- Serial, deterministic, master/slave
- Used for PLCs, sensors, actuators

**Profinet**
- Industrial Ethernet-based protocol
- Real-time, high-speed, flexible
- Used for advanced automation, robotics

**Key Features**
- Profibus: RS-485, cyclic polling, DP/PA variants
- Profinet: TCP/IP, RT/IRT classes, device discovery

**Mnemonic**: "PROcess Fieldbus, PROfessional NETwork"

**Common Use Cases**
- Factory automation, process control, distributed I/O

**Pros**
- Profibus: mature, robust, wide support
- Profinet: scalable, high performance, integrates with IT

**Cons**
- Profibus: limited speed, legacy
- Profinet: requires Ethernet infrastructure
