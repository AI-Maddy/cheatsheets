=======
OPC UA
=======

.. note::
   This is a cheatsheet for OPC UA protocol.

**OPC UA**
- Open Platform Communications Unified Architecture
- Platform-independent, service-oriented
- Used for industrial automation, IIoT

**Key Features**
- Secure, encrypted, authenticated
- Object-oriented data model
- Pub/sub and client/server
- Cross-platform, firewall-friendly

**Mnemonic**: "Open Platform, Universal Access"

**Common Use Cases**
- SCADA, MES, cloud integration, data analytics

**Pros**
- Interoperable, secure, extensible
- Rich information modeling

**Cons**
- More complex than classic OPC
- Requires more resources

**Icon**: 🛡️

**Reference**: https://opcfoundation.org/about/opc-technologies/opc-ua/
