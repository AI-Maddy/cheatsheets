============
EtherCAT
============

.. note::
   This is a cheatsheet for EtherCAT protocol.

**EtherCAT**
- Ethernet for Control Automation Technology
- Real-time industrial Ethernet protocol
- Master/slave, on-the-fly processing

**Key Features**
- Ultra-low latency, deterministic
- Daisy-chain topology, flexible wiring
- Hot connect/disconnect
- Distributed clocks for synchronization

**Mnemonic**: "Ethernet CATs chase data fast!"

**Common Use Cases**
- Motion control, robotics, high-speed automation

**Pros**
- High performance, low jitter
- Scalable, robust

**Cons**
- Requires EtherCAT-compatible hardware
- Not standard Ethernet switches
