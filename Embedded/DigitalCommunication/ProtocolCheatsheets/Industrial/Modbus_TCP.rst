============
Modbus TCP
============

.. note::
   This is a cheatsheet for Modbus TCP protocol.

**Modbus TCP**
- Ethernet-based version of Modbus
- Client/server (master/slave) model
- Used for industrial automation, SCADA

**Key Features**
- Simple, open, widely supported
- TCP/IP transport (port 502)
- Register-based data model

**Mnemonic**: "Modbus over TCP/IP"

**Common Use Cases**
- PLCs, remote I/O, energy meters

**Pros**
- Easy to implement, vendor-neutral
- Integrates with IT networks

**Cons**
- No built-in security
- Not real-time/deterministic
