=====
PPP
=====

.. note::
   This is a cheatsheet for PPP protocol.

**PPP**
- Point-to-Point Protocol
- Data link layer protocol for direct connections
- Used in dial-up, DSL, VPNs

**Key Features**
- Framing, error detection
- Authentication (PAP, CHAP)
- Link Control Protocol (LCP), Network Control Protocols (NCPs)

**Mnemonic**: "Point-to-Point Protocol"

**Common Use Cases**
- Internet dial-up, serial links, VPN tunnels

**Pros**
- Simple, supports multiple protocols
- Widely supported

**Cons**
- Limited to point-to-point
- Not for high-speed LANs
