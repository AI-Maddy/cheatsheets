=====
XMPP
=====

.. note::
   This is a cheatsheet for XMPP protocol.

**XMPP**
- Extensible Messaging and Presence Protocol
- Open, XML-based, real-time messaging
- Used for chat, presence, IoT

**Key Features**
- Decentralized, federated
- Extensible via XEPs
- Secure (TLS, SASL)

**Mnemonic**: "eXtensible Messaging and Presence Protocol"

**Common Use Cases**
- Instant messaging, IoT, collaboration tools

**Pros**
- Open, extensible, secure

**Cons**
- XML overhead
- Not optimized for high-throughput

**Icon**: 💬

**Reference**: https://xmpp.org/about/technology-overview/
