=====
HDLC
=====

.. note::
   This is a cheatsheet for HDLC protocol.

**HDLC**
- High-Level Data Link Control
- Bit-oriented, synchronous data link layer protocol
- Used in WANs, serial links, ISDN

**Key Features**
- Framing with flags, bit stuffing
- Error detection (CRC)
- Point-to-point and multipoint

**Mnemonic**: "High Data Link Control"

**Common Use Cases**
- WAN links, PPP, X.25, industrial comms

**Pros**
- Reliable, efficient, flexible

**Cons**
- Requires synchronous link
- Overhead for bit stuffing
