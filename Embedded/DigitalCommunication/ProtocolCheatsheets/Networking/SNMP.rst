=====
SNMP
=====

.. note::
   This is a cheatsheet for SNMP protocol.

**SNMP**
- Simple Network Management Protocol
- Monitors and manages network devices
- UDP port 161/162

**Key Features**
- Get/Set/Trap operations
- MIBs (Management Information Bases)
- Versions: v1, v2c, v3 (secure)

**Mnemonic**: "Simple Network Management Protocol"

**Common Use Cases**
- Network monitoring, device management

**Pros**
- Simple, widely supported

**Cons**
- Security (v1/v2c), limited by UDP
