=====
DNS
=====

.. note::
   This is a cheatsheet for DNS protocol.

**DNS**
- Domain Name System
- Resolves domain names to IP addresses
- Hierarchical, distributed database

**Key Features**
- Caching, recursive/iterative queries
- UDP (port 53), TCP for large responses
- DNSSEC for security

**Mnemonic**: "Domain Name Service"

**Common Use Cases**
- Web browsing, email, cloud services

**Pros**
- Transparent, scalable, fast

**Cons**
- Vulnerable to spoofing, DDoS
