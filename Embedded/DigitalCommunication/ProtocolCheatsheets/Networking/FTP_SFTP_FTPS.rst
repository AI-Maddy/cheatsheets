====================
FTP / SFTP / FTPS
====================

.. note::
   This is a cheatsheet for FTP, SFTP, and FTPS protocols.

**FTP**
- File Transfer Protocol
- Unencrypted, TCP port 21
- Used for file transfers, legacy systems

**SFTP**
- SSH File Transfer Protocol
- Encrypted, uses SSH (port 22)
- Secure file transfer, authentication

**FTPS**
- FTP Secure (SSL/TLS)
- Encrypted, explicit/implicit modes
- Uses ports 21/990

**Key Features**
- FTP: simple, legacy
- SFTP/FTPS: secure, modern

**Mnemonic**: "File Transfer, Securely!"

**Common Use Cases**
- Website deployment, backups, data exchange

**Pros**
- SFTP/FTPS: secure, widely supported

**Cons**
- FTP: insecure, deprecated
- SFTP/FTPS: firewall/NAT issues
