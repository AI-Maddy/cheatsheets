=====
BGP
=====

.. note::
   This is a cheatsheet for BGP protocol.

**BGP**
- Border Gateway Protocol
- Path vector routing protocol
- Used for Internet backbone, ISPs

**Key Features**
- Inter-domain routing (between ASes)
- Path attributes, policies
- TCP port 179

**Mnemonic**: "Border Gateway Protocol"

**Common Use Cases**
- Internet routing, large networks

**Pros**
- Scalable, policy-based

**Cons**
- Complex, slow convergence
