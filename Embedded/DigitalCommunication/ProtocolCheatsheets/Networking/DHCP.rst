=====
DHCP
=====

.. note::
   This is a cheatsheet for DHCP protocol.

**DHCP**
- Dynamic Host Configuration Protocol
- Assigns IP addresses automatically
- UDP port 67/68

**Key Features**
- Lease, renew, release
- Options for DNS, gateway, etc.
- DORA process (Discover, Offer, Request, Ack)

**Mnemonic**: "Dynamic Host Configuration Protocol"

**Common Use Cases**
- Home, enterprise, cloud networks

**Pros**
- Plug-and-play, reduces admin

**Cons**
- Security risks (rogue servers)

**Icon**: 🏠
