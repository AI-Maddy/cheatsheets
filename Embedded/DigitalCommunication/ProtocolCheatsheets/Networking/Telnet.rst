=======
Telnet
=======

.. note::
   This is a cheatsheet for Telnet protocol.

**Telnet**
- Legacy remote access protocol
- Unencrypted, TCP port 23
- Used for device management, testing

**Key Features**
- Simple, text-based
- No encryption
- Replaced by SSH for security

**Mnemonic**: "Tell the Net!"

**Common Use Cases**
- Legacy device access, network testing

**Pros**
- Simple, universal

**Cons**
- Insecure, deprecated

**Icon**: 🖥️
