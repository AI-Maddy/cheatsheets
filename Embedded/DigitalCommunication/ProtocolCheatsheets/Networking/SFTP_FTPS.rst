============
SFTP / FTPS
============

.. note::
   This is a cheatsheet for SFTP and FTPS protocols.

**SFTP**
- SSH File Transfer Protocol
- Secure, uses SSH (port 22)
- File transfer, authentication

**FTPS**
- FTP Secure (SSL/TLS)
- Encrypted, explicit/implicit modes
- Uses ports 21/990

**Key Features**
- Secure file transfer
- Authentication, encryption

**Mnemonic**: "Secure File Transfer Protocols"

**Common Use Cases**
- Secure file transfer, backups

**Pros**
- Secure, widely supported

**Cons**
- Firewall/NAT issues
