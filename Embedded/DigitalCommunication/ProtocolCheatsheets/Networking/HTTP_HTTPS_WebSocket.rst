====================
HTTP / HTTPS / WebSocket
====================

.. note::
   This is a cheatsheet for HTTP, HTTPS, and WebSocket protocols.

**HTTP**
- Hypertext Transfer Protocol
- Stateless, request/response
- TCP port 80

**HTTPS**
- HTTP Secure (TLS/SSL)
- Encrypted, secure
- TCP port 443

**WebSocket**
- Persistent, full-duplex communication
- Used for real-time web apps
- TCP port 80/443 (upgrade from HTTP)

**Key Features**
- HTTP: simple, extensible
- HTTPS: secure, trusted
- WebSocket: low-latency, bidirectional

**Mnemonic**: "Web Protocols: Secure & Live!"

**Common Use Cases**
- Web browsing, APIs, chat, gaming

**Pros**
- Universal, open standards
- HTTPS: privacy, integrity

**Cons**
- HTTP: insecure
- WebSocket: firewall/NAT issues
