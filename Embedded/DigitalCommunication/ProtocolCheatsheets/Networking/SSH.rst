=====
SSH
=====

.. note::
   This is a cheatsheet for SSH protocol.

**SSH**
- Secure Shell
- Secure remote access, file transfer
- TCP port 22

**Key Features**
- Encrypted, authenticated
- Tunneling, port forwarding
- Key-based authentication

**Mnemonic**: "Secure SHell"

**Common Use Cases**
- Server management, SFTP, tunneling

**Pros**
- Secure, flexible, widely supported

**Cons**
- Requires key management

**Icon**: 🗝️

**Reference**: https://en.wikipedia.org/wiki/Secure_Shell
