====================
TCP / UDP / IP
====================

.. note::
   This is a cheatsheet for TCP, UDP, and IP protocols.

**TCP**
- Transmission Control Protocol
- Reliable, connection-oriented
- Used for web, email, file transfer

**UDP**
- User Datagram Protocol
- Fast, connectionless, no guarantee
- Used for streaming, gaming, VoIP

**IP**
- Internet Protocol (IPv4/IPv6)
- Packet delivery, addressing
- Foundation of the Internet

**Key Features**
- TCP: retransmission, flow control
- UDP: low latency, no overhead
- IP: routing, fragmentation

**Mnemonic**: "Three Core Internet Protocols"

**Common Use Cases**
- Web, streaming, gaming, IoT

**Pros**
- Universal, open standards

**Cons**
- UDP: no reliability
- TCP: overhead, slower
