SOME/IP
=======

**Domain:** Automotive

**Description:**
Service-oriented Ethernet communication protocol.

	<div style="background-color:#e3f2fd; border-radius:10px; padding:16px; margin-bottom:16px;">

**Key Features:**
- 🌐 Service-oriented, scalable
- 🔄 Runs over Ethernet/IP
- 🛠️ Enables modular, updatable vehicle software
- 🧩 Used for infotainment, ADAS, diagnostics

**Physical/Electrical Layer:**
- 🔌 Ethernet (100BASE-T1, 1000BASE-T1)
- ⚡ Data rates: 100 Mbps, 1 Gbps

**Topology:**
- 🌟 Switched Ethernet

**Use Cases:**
- 🚗 Infotainment, ADAS, diagnostics
- 🛠️ Over-the-air updates, modular software

**Pros:**
- 🚀 Flexible, scalable, future-proof
- 🧩 Integrates with AUTOSAR

**Cons:**
- 🏗️ Complex, requires Ethernet backbone
- ⚠️ Security is critical

**Standard:**
- 📜 AUTOSAR, ISO 17215

**Typical Applications:**
- 🚗 Modern vehicles, especially with advanced infotainment

**Memorization Tip:**
<span style="color:#1976d2;">Think of SOME/IP as "services on demand"—your car's apps, delivered over Ethernet!</span>
