USB
===

**Domain:** Embedded/General

**Description:**
USB (Universal Serial Bus) is a widely used standard for connecting peripherals to computers and embedded systems, supporting hot-plugging, power delivery, and high-speed data transfer.

**Key Features:**
- 🖥️ Hot-pluggable, plug-and-play
- 🔄 Supports multiple device classes
- ⚡ Power delivery and charging
- 🚀 High-speed data transfer (up to 40 Gbps for USB4)

**Physical/Electrical Layer:**
- 🔌 Differential signaling, shielded cable
- ⚡ Data rates: 1.5 Mbps (Low), 12 Mbps (Full), 480 Mbps (High), 5/10/20/40 Gbps (SuperSpeed/USB4)

**Topology:**
- 🌟 Star (host with multiple devices)

**Use Cases:**
- 🖥️ Keyboards, mice, storage, cameras
- 🛠️ Embedded systems, charging

**Pros:**
- 🚀 Fast, flexible, widely supported
- 🔌 Power and data in one cable

**Cons:**
- 🏗️ Complex protocol stack
- ⚠️ Cable/connector confusion

**Standard:**
- 📜 USB-IF, USB 1.1/2.0/3.x/4.0

**Typical Applications:**
- 🖥️ PCs, laptops, embedded devices

**Memorization Tip:**
<span style="color:#1976d2;">Think of USB as the "universal plug"—one cable for everything, everywhere!</span>
