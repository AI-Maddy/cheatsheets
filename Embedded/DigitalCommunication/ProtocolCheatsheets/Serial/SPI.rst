SPI
===

**Domain:** Embedded/Industrial

**Description:**
High-speed, 4-wire serial bus.

.. raw:: html

	<div style="background-color:#e3f2fd; border-radius:10px; padding:16px; margin-bottom:16px;">
	<h2 style="color:#1565c0;">🔌⚡ SPI (Serial Peripheral Interface)</h2>
	<b>Domain:</b> <span style="color:#1976d2;">Embedded/Industrial</span><br>
	<b>Mnemonic:</b> <span style="color:#388e3c;">"Speedy Peripheral Interconnect"</span><br>
	<b>Icon:</b> 🔌⚡<br>
	</div>

**Key Features:**
- ⚡ High-speed, synchronous serial
- 🔄 4-wire (MOSI, MISO, SCLK, SS)
- 🧩 Supports multiple slaves
- 🛡️ Full-duplex communication

**Physical/Electrical Layer:**
- 🔌 Push-pull, TTL/CMOS logic
- ⚡ Data rates: up to tens of Mbps

**Topology:**
- 🌟 Star (master with multiple slaves)

**Use Cases:**
- 🖥️ Flash memory, sensors, displays
- 🛠️ Board-level, high-speed comms

**Pros:**
- 🚀 Fast, simple, low latency
- 🧩 Easy to implement

**Cons:**
- 🏗️ More wires than I²C
- ⚠️ No formal standard, vendor variations

**Standard:**
- 📜 De facto, vendor-specific

**Typical Applications:**
- 🖥️ Microcontrollers, embedded systems

**Memorization Tip:**
<span style="color:#1976d2;">Think of SPI as the "speedy 4-lane highway" for chips—fast, direct, and always in sync!</span>
